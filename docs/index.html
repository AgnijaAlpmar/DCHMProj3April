Description:  

This project is made for … by… as an attempt for digitization of building plan drawings of Drottningholms palace theatre. It started as an interest to find a one particular plan… from building extension from year.. and turned to a bunch of plan drawings from the early stage of the planning for the current theatre to be built on the ruins of the unluckelly burnt down theatre standing there before.
